**README - SDK de Culqi**

Este repositorio contiene el SDK de Culqi para PHP. 
Sigue los siguientes pasos para levantar el proyecto:

### Instalar Python3

```bash
    # instalar Python3
    sudo apt update
    sudo apt install python3 python3-pip
```

### Instalar Dependencias
```bash
    pip install -r requirements.txt
```
